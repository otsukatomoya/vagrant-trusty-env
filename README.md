# vagrant-trusty-env

## Outline

### Environment
* Ubuntu 14.04 Trusty
* Nginx
* Ruby
* Ruby on Rails
* Node.js
* MySQL


## Introduction

### Available Box
```
$ vagrant box add trusty https://cloud-images.ubuntu.com/vagrant/trusty/current/trusty-server-cloudimg-amd64-vagrant-disk1.box
```

### Vaagrant Up
```
$ cd ~/.../vagrant-trusty-env
$ vagrant up
```
